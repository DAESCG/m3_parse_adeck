# List of Models used in ADECKs

### [ Full Document at RAL
](http://www.ral.ucar.edu/hurricanes/repository/techlist/index.php)

This section attempts to provide a comprehensive and searchable listing of the
TECH designators of all TC guidance aids, both past and present. Links provide
more information about a particular model, including which organization
contributed, and documentation about that model.

Jump links for the various categories of TECH identifiers:

  * Official Warnings and ATCF Information
  * Operational Models - Track Only
  * Experimental Models - Track Only
  * Operational Models - Intensity Only
  * Experimental Models - Intensity Only
  * Operational Models - Both Track and Intensity
  * Experimental Models - Both Track and Intensity
  * Operational Consensus Aids - Track Only
  * Experimental Consensus Aids - Track Only
  * Operational Consensus Aids - Intensity Only
  * Experimental Consensus Aids - Intensity Only
  * Experimental Consensus Aids - Both Track and Intensity
  * Operational Wind Radii Models
  * Deprecated Models

### Data Sources

This list was compiled from the following sources:

  * NHC's techlist of all _current_ TECH's that can appear in NHC's public a-decks:  
[ ftp://ftp.nhc.noaa.gov/atcf/docs/nhc_techlist.dat](ftp://ftp.nhc.noaa.gov/at
cf/docs/nhc_techlist.dat)

  * NHC's techlist of all _deprecated _TECH's that no longer appear in NHC's public a-decks:  
[ ftp://ftp.nhc.noaa.gov/atcf/docs/nhc_techlist.dat.deprecated](ftp://ftp.nhc.
noaa.gov/atcf/docs/nhc_techlist.dat.deprecated)

  * JTWC's techlist of all _current_ TECH's that can appear in JTWC's a-decks (not public):  
[http://www.nrlmry.navy.mil/atcf_web/docs/database/new/techlist.dat.pearl
](http://www.nrlmry.navy.mil/atcf_web/docs/database/new/techlist.dat.pearl)

  * The Tropical Cyclone Modeling Team (TCMT)'s [ 2012 HFIP Demonstration Page](http://www.ral.ucar.edu/projects/hfip/d2012/models/regional.php) \- regional models section
